## Visual Studio Hotkeys
Formatting: https://stackoverflow.com/questions/4942113/is-there-a-format-code-shortcut-for-visual-studio

Two-step hotkey:
- Ctrl + E, THEN Ctrl + D (document) OR Ctrl + F (selection)
